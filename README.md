## О проекте (Tyrin Sergey)

**Система бронирования переговорных комнат с учётом конфликтов** — это RESTful веб-сервис для автоматизации бронирования переговорных комнат в офисах компаний.

### Проблема, которую решает проект
- ❌ Двойное бронирование комнат
- ❌ Отсутствие прозрачности занятости помещений
- ❌ Невозможность анализировать загруженность
- ❌ Конфликты при согласовании времени

### Целевая аудитория
- 🏢 Сотрудники компаний — для быстрого бронирования
- 👔 Менеджеры и руководители — для контроля загруженности
- 🔧 Администраторы офиса — для оптимизации использования комнат

## Основной функционал

### 1. Управление пользователями
- Регистрация новых пользователей
- Аутентификация по JWT токенам
- Защита эндпоинтов авторизацией

### 2. Управление переговорными комнатами
- CRUD операции (создание, чтение, обновление, удаление)
- Просмотр всех активных комнат
- Мягкое удаление комнат

### 3. Управление бронированиями
- Создание бронирований с автоматической проверкой конфликтов
- Просмотр своих бронирований
- Обновление и отмена бронирований

### 4. Аналитика и алгоритмические задачи
- Проверка конфликтов временных интервалов
- Расчёт загруженности комнат за 30 дней
- Рекомендации по оптимизации использования
- Персональная статистика пользователей

## Структура базы данных

### Таблицы

| Таблица | Описание | Связи |
|---------|----------|-------|
| **users** | Пользователи системы | Один ко многим с bookings |
| **rooms** | Переговорные комнаты | Один ко многим с bookings |
| **bookings** | Бронирования | ForeignKey на users и rooms |


## Установка и запуск

### Требования
- Python 3.14 или выше
- Git

### Инструкция по установке

# 1. Клонирование репозитория
git clone https://github.com/ваш-username/booking-system.git
cd booking-system

# 2. Создание виртуального окружения
python -m venv venv

# 3. Активация виртуального окружения
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 4. Установка зависимостей
pip install -r requirements.txt
pip install sqlalchemy aiosqlite -i https://mirrors.cloud.tencent.com/pypi/simple
pip install pydantic[email]

# 5. Запуск приложения
python run.py

После запуска сервис будет доступен по адресу: http://127.0.0.1:8000
